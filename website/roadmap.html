// Six-month build plan
THE ROAD
AHEAD

Each phase ships in a live environment — no hidden waitlists, no vaporware. If it's on this list, it gets built or it gets cut.

Months 1–2 · Phase 01 ⟳ IN PROGRESS
Speed & Execution

Closing the loop from signal to trade. Tighter conviction scoring so you know what to act on and how fast.

  • ▸
    Alert auto-trading (beta) — optional execution layer on token alerts with paper mode, hard size caps, max daily loss, and a big red off switch. Exchange + RPC connectors added per chain.
  • ▸
    Spark → Pulse → Surge tiers — tiered HVA activity scoring on fresh (<7d) vs seasoned accounts. Same statistical pattern as fast-attention tools, calibrated to your curated HVA graph.
  • ▸
    Alpha Profile Card — one lookup: X bio, project history, stored handle variants, verified metrics, and archive research link. Honest about what X's API exposes.
  • ▸
    Bring-your-own wallet labels — import CSV or sync-style lists so addresses in alerts show your own names, like a personal GMGN/explorer overlay.
Months 3–4 · Phase 02 COMING SOON
Depth & On-chain

Moving from surface signals to sustained conviction and chain-level data. Fewer false positives, more context per alert.

  • ▸
    Conviction clock — rolling HVA engagement over 24h / 3d / 7d on the same project handle. Slow-burn detection, not just spike chasing.
  • ▸
    Mention velocity — how fast curated accounts mention or quote-tweet a ticker before liquidity catches up. Early hype radar, separate signal stream.
  • ▸
    Solana token panel (lite) — compact SOL pair view next to alerts: liquidity snapshot, holder concentration stub, DexScreener MC cross-check. Not a full terminal — a power sidebar.
  • ▸
    Unified alert routing — per-user rules: which signals hit Discord vs web vs push; mute categories; VIP-HVA-only mode without touching bot config.
Month 5 · Phase 03 COMING SOON
Intelligence Layer

Surface what the data already knows. Internal boards, narrative detection, and smart summaries — all opt-in, all labeled.

  • ▸
    Trenches board — internal 24h leaderboard mixing mentions, new followers, and alert velocity. Our "trending smart accounts" view built from data we already collect, no third-party feed dependency.
  • ▸
    Elite desk — a second, stricter HVA tier for the highest-signal callers. Separate alert stream, separate scoring, separate conviction thresholds.
  • ▸
    AI project digests (opt-in) — one-paragraph "why now" on project detail pages. Strictly labeled, human-editable by admins, off by default for users who prefer raw data.
Month 6 · Phase 04 COMING SOON
Scale & Polish

Execution graduates from beta. Presets, fee routing, and cross-link intelligence for power users.

  • ▸
    Strategy presets — save alert → trade templates with size, venue, and time-in-force. Fee-aware routing when auto-trading is stable on mainnet.
  • ▸
    Smart money flow (lite) — in/out snapshots per token where RPC budget allows. Not real-time order flow — directional pressure read from on-chain aggregates.
  • ▸
    Cross-link hints — when users supply wallet ↔ handle pairs, surface them responsibly on profile cards. Privacy-first, no doxxing agenda.
  • ▸
    Fragment resolver — paste a truncated address or tweet URL, expand to full pubkey when possible. Utility-focused, no magic, no data we don't have.
